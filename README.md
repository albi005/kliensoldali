# TodoReact

Teendőkezelő React alkalmazás, passkey bejelentkezéssel és push értesítésekkel.

Az alkalmazás segítségével teendőket lehet létrehozni, azokhoz címet, határidőt és leírást (markdown formátumban) megadni, a teendőkhöz emlékeztetőket létrehozni, amelyekről az oldal automatikusan értesítést küld Web Push segítségével.

Használható bejelentkezés nélkül, ilyenkor az adatok a felhasználó böngészőjében tárolódnak, vagy azonsítókulcsos bejelentkezzésel, ilyenkor az alkalmazás mögötti szerver tárolja az adatokat. Egy felhasználó több azonosítókulcsot is hozzáadhat a fiókjához.
